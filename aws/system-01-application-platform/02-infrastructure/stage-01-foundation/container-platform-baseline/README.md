# Container Platform Baseline

This module defines the baseline infrastructure for the Tier-1 AWS Application Platform.

## Purpose

This stage establishes foundational infrastructure components used for platform experimentation and validation.

## Current Resources

- S3 bucket used for baseline infrastructure validation

## Terraform Files

- versions.tf → Terraform and provider version requirements
- provider.tf → AWS provider configuration
- variables.tf → Input variables
- main.tf → Infrastructure resources
- outputs.tf → Output values after deployment
